# Java in VSCode

## Create a new Java Project

- `Ctrl + Shift + P`
- Search for Java project
- No build tools
- Select project location
- Name the project
- Create

## Create getters and setters

- Right click
- Source action
- Generate getters/setters
- Select variables
- Click `OK`

## Override toString

- Right click
- Source action
- Override methods
- Select method
- Click `OK`
